# Tech Stack

## Frontend Architecture

- **Language:** HTML5, CSS3, and modern Vanilla JavaScript (ES6+).
- **Core Strategy:** Direct DOM manipulation for a minimalist and efficient user experience. No heavy frontend frameworks.

## Styling & Layout

- **Tailwind CSS:** Integrated via CDN for rapid, responsive design and utility-first styling without the need for a complex build pipeline.
- **Custom CSS:** Minimal custom styles to handle specific traditionally-inspired UI elements and Islamic motifs.

## Logic & State

- **Native Calculation Logic:** Fidyah calculation functions written in pure JavaScript.
- **State Management:** Simple component-level state managed through native JavaScript variables and object properties.

## Build & Tooling

- **Build Step:** None. The project is designed to be served directly from the file system or any static web server.
- **Development Server:** Standard browser-based preview or a simple local server (e.g., VS Code Live Server).

## Deployment

- **Target Platform:** Any static hosting provider (e.g., GitHub Pages, Netlify, or Vercel).
