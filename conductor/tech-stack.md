# Tech Stack\n\n## Frontend Architecture\n- **Language:** HTML5, CSS3, and modern Vanilla JavaScript (ES6+).\n- **Core Strategy:** Direct DOM manipulation for a minimalist and efficient user experience. No heavy frontend frameworks.\n\n## Styling & Layout\n- **Tailwind CSS:** Integrated via CDN for rapid, responsive design and utility-first styling without the need for a complex build pipeline.\n- **Custom CSS:** Minimal custom styles to handle specific traditionally-inspired UI elements and Islamic motifs.\n\n## Logic & State\n- **Native Calculation Logic:** Fidyah calculation functions written in pure JavaScript.\n- **State Management:** Simple component-level state managed through native JavaScript variables and object properties.\n\n## Build & Tooling\n- **Build Step:** None. The project is designed to be served directly from the file system or any static web server.\n- **Development Server:** Standard browser-based preview or a simple local server (e.g., VS Code Live Server).\n\n## Deployment\n- **Target Platform:** Any static hosting provider (e.g., GitHub Pages, Netlify, or Vercel).
