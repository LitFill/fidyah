# Track Specification: Core Fidyah Calculator

## Overview

Build a static fidyah calculator using Vanilla JS and Tailwind CSS, adhering to the Syafii school of fiqh.

## Requirements

- **User Stories:** Basic calculation, multi-year doubling, educational explanations.
- **Functional Reqs:** Fidyah math (Days x Years x 1 Mudd), multi-Qoul weight conversion (0.675kg, 0.75kg, 0.8kg), cost estimation.
- **Non-Functional Reqs:** High performance, accessibility, zero backend dependency.
- **Edge Cases:** Large year input (10+ years), exemption logic.

## Tech Stack

- Vanilla HTML/JS/CSS
- Tailwind CSS (CDN)
- No external build step
