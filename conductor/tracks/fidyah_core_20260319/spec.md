# Track Specification: Core Fidyah Calculator\n\n## Overview\nBuild a static fidyah calculator using Vanilla JS and Tailwind CSS, adhering to the Syafii school of fiqh.\n\n## Requirements\n- **User Stories:** Basic calculation, multi-year doubling, educational explanations.\n- **Functional Reqs:** Fidyah math (Days x Years x 1 Mudd), multi-Qoul weight conversion (0.675kg, 0.75kg, 0.8kg), cost estimation.\n- **Non-Functional Reqs:** High performance, accessibility, zero backend dependency.\n- **Edge Cases:** Large year input (10+ years), exemption logic.\n\n## Tech Stack\n- Vanilla HTML/JS/CSS\n- Tailwind CSS (CDN)\n- No external build step
