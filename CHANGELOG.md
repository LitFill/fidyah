# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-19

### Added

#### Core Features
- Fidyah calculator based on Madzhab Syafii
- Multi-year fidyah doubling calculation (Days × Ramadhan elapsed)
- Multi-Qoul mudd conversion (0.675kg, 0.75kg, 0.8kg)
- Cost estimation based on user-provided rice price
- Real-time calculation updates

#### UI/UX
- Clean, responsive design using Tailwind CSS
- Educational explanations (dalil) for each calculation step
- Accessibility features:
  - Large font mode toggle
  - High contrast mode toggle
  - Skip link for keyboard navigation
  - ARIA labels and descriptions
- Share/download result functionality:
  - Copy to clipboard
  - Download as .txt file

#### Documentation
- README.md with project overview
- USAGE.md with detailed usage guide
- FAQ.md with comprehensive Q&A
- CONTRIBUTING.md with contribution guidelines

### Deployment
- GitHub Pages deployment
- Live site: https://litfill.github.io/fidyah/
